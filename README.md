# 3D Developer Portfolio

This is a modern, animated, and responsive 3D developer portfolio built with **React**, **Three.js (via @react-three/fiber)**, and **Framer Motion**. It showcases projects, skills, work experience, and contact details with a sleek dark-themed design.

## 🔥 Features

- ✨ Interactive 3D model on the homepage
- 📁 Projects with descriptions and GitHub links
- 📜 Timeline-based work experience
- 🛠️ Skills and tools overview
- 📨 Contact form with email integration
- 💬 Testimonials (optional)
- 📱 Fully responsive design

## 🧰 Tech Stack

- React.js
- Tailwind CSS
- Three.js (`@react-three/fiber`, `@react-three/drei`)
- Framer Motion
- EmailJS (for contact form)

## 🚀 Deployment

This project is deployed on **Vercel** for fast global performance and easy CI/CD integration.
